# DARK013TATTOO - Gestão Profissional (Cloud)

Sistema de gestão para estúdio de tatuagem e piercing com sincronização na nuvem via **Supabase**.  
Permite controle de caixa, serviços, agenda, estoque de piercings, materiais e relatórios gerenciais.

## 🚀 Funcionalidades

- Dashboard com indicadores financeiros e gráficos
- Fluxo de caixa (entradas/saídas, saldo)
- Cadastro de serviços (tatuagem/piercing) com cálculo de repasse (30% estúdio / 70% artista)
- Agenda de atendimentos com status (agendado, confirmado, concluído, cancelado)
- Estoque de piercings e registro de vendas
- Estoque de materiais e controle de uso
- Relatórios por tatuador e lucro líquido
- Backup/restauração dos dados em JSON

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript (ES6+)
- [Supabase](https://supabase.com/) (banco de dados e API)
- [Chart.js](https://www.chartjs.org/) (gráficos)
- [Font Awesome](https://fontawesome.com/) (ícones)
