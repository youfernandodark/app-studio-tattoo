# DARK013TATTOO - Sistema de Gestão

Sistema completo de gestão para estúdio de tatuagem com interface dark mode.

## 🚀 Funcionalidades

- **Dashboard**: Visão geral do negócio
- **Fluxo de Caixa**: Controle de entradas e saídas
- **Serviços**: Registro de tatuagens e piercings realizados
- **Piercings**: Controle de estoque e revenda
- **Materiais**: Gestão de materiais de consumo
- **Agenda**: Agendamento de clientes
- **Relatórios**: Relatórios gerenciais completos
- **Backup**: Exportação e importação de dados
